Evaluation of Common ML Models Against Tabular Numeric Data

This project contains a notebook that is useful as a first pass for any tabular binary classification or regression problem. It compares a number of ML models and does a feature analysis to provide insight into most impactful features within data-set. To view the easy to follow Jupyter notebook I have create follow this link.

Models Compared

  • Logistic Regression
  • K Nearest Neighbor
  • Support Vector Machine
  • Gaussian Naive Bayes
  • Random Forrest Classifier
  • Stacking Classifier (all above)
  • XGBoost
  • MLP Classifier
